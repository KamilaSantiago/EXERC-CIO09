Aplicação prática

Utilizaria para controlar o layout de elementos em uma página web.

  • Display: Define o tipo de caixa que um elemento HTML cria.
  • Flex-direction: Especifica a direção em que os itens flexíveis são dispostos dentro do contêiner flexível;
  • Justify-content: Define como os itens flexíveis são distribuídos ao longo do eixo principal.
  • Align-items: Define como os itens flexíveis são alinhados ao longo do eixo transversal.
